# Gantry

![](<assets/Assembly%20(2).png>)

DAVID's gantry is based on 3 vertical running MGN12H linear rails with 30 degree bearing mounts to hold printed arms out on the sides.

![](assets/Arm.png)
This is the secret to the arms being rigid: a hole in the middle. Normally, printed parts are pretty flimsy, especially when they get long and thin like these. I'll be pausing the print partway through and gluing a piece of steel into the middle of the print, making it significantly more rigid. If need be, I might do this for the toolhead too.
